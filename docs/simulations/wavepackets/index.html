About this simulation

This is a demonstration of wave interference. You can imagine that these waves are sound waves, light waves, water waves, quantum wave functions, or anything else that's a wave really. When two waves overlap with each other in space, the resulting wave is a sum of both waves at every point. This is known as superposition.

The resultant wave is drawn using a thick red line, while the individual waves that sum to make it are drawn in a thin green line. You can select a wave using the dropdown menu and change its parameters with the sliders.

Here are some examples to try

  1. Two identical waves, but one travelling in the reverse direction. This forms a standing wave, like what forms on a guitar string.
  2. Two waves with a different phase offset. Notice that when the offset is exactly 180 degrees (or pi radians), the waves are in antiphase and cancel each other out. It's by this principle that noise cancelling headphones work!
  3. Multiple waves, each with a very slightly different wavelength. This creates a wave packet, which appears to be a short wavelength wave with a long wavelength envelope. When talking about sound, this is known as a beat. The long wavelength envelope can actually be heard, either as a variation in volume, or as a note itself if its frequency is high enough to be audiable. This works even when one of the waves is travelling in the reverse direction.
